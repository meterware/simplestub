Using ASM instead of Javassist

Simplestub is built using the Javassist library; however, some platforms (such as Android) do not support the features it needs. An alternative is to use the simplestub-asm extension:

In Maven, this means adding the following to your project:

    <dependency>
        <group>com.meterware.simplestub</group>
        <artifact>simplestub-asm</artifact>
        <version>1.2.0</version>
        <scope>test</scope>
    </dependency>

This will, in turn, pull in asm, asm-tree, and asm-commons
